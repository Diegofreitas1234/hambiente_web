
# 📝 Lista de Tarefas

Projeto simples de **Lista de Tarefas (To-Do List)** desenvolvido com **HTML, CSS e JavaScript**.

O objetivo do projeto é praticar manipulação do DOM e eventos em JavaScript.

---

## 🚀 Funcionalidades

- ➕ Adicionar novas tarefas
- 🗑️ Remover tarefas da lista
- ⚠️ Validação para evitar tarefas vazias

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
